1. Consulta de Documentos y Proyecciones

En la clase anterior aprendimos cómo estructurar colecciones y cómo insertar documentos en MongoDB. Ahora, estudiaremos cómo recuperar esa información de manera precisa mediante las operaciones de lectura, cuyo pilar principal es el método find().

Sintaxis Básica:
db.coleccion.find(filtro, proyeccion)
El Filtro (Query)

Es un documento que especifica los criterios de selección. Si está vacío ({}), MongoDB retornará todos los documentos de la colección.

Ejemplo: db.productos.find({ categoria: "Tecnología" }).

La Proyección (Projection)

Determina qué campos del documento deben ser devueltos al cliente. Permite ahorrar ancho de banda y memoria al filtrar datos innecesarios.

  • 1 para incluir un campo.
  • 0 para excluir un campo (por defecto, _id siempre se incluye a menos que se excluya explícitamente con 0).

Ejemplo: db.productos.find({}, { nombre: 1, precio: 1, _id: 0 }).

2. Operadores de Comparación y Lógicos

Para realizar consultas complejas más allá de una igualdad simple, MongoDB provee operadores especiales que comienzan con el símbolo de dólar ($).

Operadores de Comparación
Operador Significado Ejemplo
$eq Igual a { precio: { $eq: 100 } }
$ne No igual a { categoria: { $ne: "Libros" } }
$gt Mayor que { precio: { $gt: 50 } }
$gte Mayor o igual que { stock: { $gte: 10 } }
$lt Menor que { precio: { $lt: 20 } }
$lte Menor o igual que { stock: { $lte: 5 } }
$in Contenido en un array { categoria: { $in: ["Hogar", "Jardín"] } }
Operadores Lógicos
Operador Descripción Uso típico
$and Une condiciones con un AND lógico. Se usa implícitamente al separar por comas. { $and: [{ precio: { $gt: 10 } }, { stock: { $gt: 0 } }] }
$or Une condiciones con un OR lógico. Al menos una condición debe cumplirse. { $or: [{ categoria: "Juguetes" }, { precio: { $lt: 15 } }] }
$not Invierte el efecto de una expresión de filtro. { precio: { $not: { $gt: 50 } } }

Tip: Por defecto, si declaras varias propiedades separadas por comas dentro de un objeto de filtro, MongoDB aplica un $and implícito: { precio: { $gt: 50 }, stock: { $gt: 0 } }.
