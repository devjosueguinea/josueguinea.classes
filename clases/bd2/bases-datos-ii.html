Bases de Datos II

MongoDB, Cassandra, consumo de API

Descripción de la materia

Esta materia se enfoca en bases de datos NoSQL, incluyendo MongoDB y Cassandra, así como el consumo de APIs para la integración de datos en aplicaciones modernas.
