Clave (Key) String Número Booleano Arreglo Objeto
JSON sin Arreglos
  • Solo pares clave: valor simples
  • Valores: texto, número, booleano u objeto
  • Ideal para representar un solo documento
  • Estructura plana, directa y fácil de leer
JSON con Arreglos
  • Contiene valores del tipo array [ ]
  • Permite agrupar múltiples valores en un campo
  • Ideal para colecciones MongoDB con listas
  • Mayor expresividad y jerarquía de datos
